5. Système de Messagerie

Architecture

┌────────────────────────────────────────────────────────────────┐ │ CONVERSATION │ ├────────────────────────────────────────────────────────────────┤ │ • Lien entre 1 Locataire et 1 Centre Commercial │ │ • Peut être liée à une Réservation (optionnel) │ │ • Sujet optionnel │ │ • Peut être archivée │ └────────────────────────────────────────────────────────────────┘ │ │ 1:N ▼ ┌────────────────────────────────────────────────────────────────┐ │ MESSAGE │ ├────────────────────────────────────────────────────────────────┤ │ • contenu (TEXT) │ │ • dateEnvoi (datetime) │ │ • estLu (boolean) - marqué lu quand l'autre partie lit │ │ • dateLecture (datetime, nullable) │ │ • typeExpediteur ('locataire' ou 'centre') │ └────────────────────────────────────────────────────────────────┘

Logique de Lecture des Messages

Règle importante : Un message est considéré "non lu" pour le locataire si :
  • estLu = false
  • typeExpediteur = 'centre' (envoyé par le centre, pas par le locataire lui-même)

Marquage Automatique comme Lu

src/Controller/Locataire/MessageController.php
#[Route('/dashboard/locataire/messages/{id}', name: 'locataire_message_show')] public function show( Conversation $conversation, EntityManagerInterface $em ): Response { // Vérifier que le locataire est bien participant if ($conversation->getLocataire() !== $this->getUser()) { throw $this->createAccessDeniedException(); } // Marquer tous les messages du CENTRE comme lus foreach ($conversation->getMessages() as $message) { if ($message->getTypeExpediteur() === 'centre' && !$message->isEstLu()) { $message->setEstLu(true); $message->setDateLecture(new \DateTime()); } } $em->flush(); return $this->render('locataire/message/show.html.twig', [ 'conversation' => $conversation, ]); }

Envoi d'un Message avec CSRF

src/Controller/Locataire/MessageController.php
#[Route('/dashboard/locataire/messages/{id}/envoyer', name: 'locataire_message_send', methods: ['POST'])] public function send( Request $request, Conversation $conversation, EntityManagerInterface $em ): Response { // Validation CSRF if (!$this->isCsrfTokenValid('send_message'.$conversation->getId(), $request->request->get('_token'))) { throw $this->createAccessDeniedException('Token CSRF invalide'); } $contenu = $request->request->get('contenu'); if (!empty($contenu)) { $message = new Message(); $message->setConversation($conversation); $message->setContenu($contenu); $message->setDateEnvoi(new \DateTime()); $message->setEstLu(false); $message->setTypeExpediteur('locataire'); // Mettre à jour la date du dernier message $conversation->setDernierMessageDate(new \DateTime()); $em->persist($message); $em->flush(); } return $this->redirectToRoute('locataire_message_show', ['id' => $conversation->getId()]); }

6. Gestion des Réservations

Système de Filtres

La page des réservations supporte 9 filtres différents :

Filtre Critère Description
toutes Aucun Toutes les réservations
en_attente statut = EN_ATTENTE En attente de validation
validee statut = VALIDEE Validées, avant début
en_cours statut = EN_COURS Actuellement actives
terminee statut = TERMINEE Passées et finies
refusee statut = REFUSEE Refusées par le centre
annulee statut = ANNULEE Annulées
actives EN_ATTENTE ou EN_COURS + dates valides Réservations en cours ou à venir
a_venir dateDebut > maintenant Réservations futures
passees dateFin < maintenant Réservations passées

#[Route('/favoris/ajouter/{id}', name: 'locataire_favori_ajouter', methods: ['POST'])] public function ajouter( Emplacement $emplacement, FavoriRepository $favoriRepo, EntityManagerInterface $em ): JsonResponse { $locataire = $this->getUser(); // Vérifier si déjà en favori $existant = $favoriRepo->findOneBy([ 'locataire' => $locataire, 'emplacement' => $emplacement ]); if ($existant) { return new JsonResponse([ 'success' => false, 'message' => 'Déjà en favoris' ], 400); } $favori = new Favori(); $favori->setLocataire($locataire); $favori->setEmplacement($emplacement); $favori->setDateAjout(new \DateTimeImmutable()); $em->persist($favori); $em->flush(); return new JsonResponse(['success' => true]); } #[Route('/favoris/retirer/{id}', name: 'locataire_favori_retirer', methods: ['POST'])] public function retirer( Emplacement $emplacement, FavoriRepository $favoriRepo, EntityManagerInterface $em ): JsonResponse { $locataire = $this->getUser(); $favori = $favoriRepo->findOneBy([ 'locataire' => $locataire, 'emplacement' => $emplacement ]); if ($favori) { $em->remove($favori); $em->flush(); } return new JsonResponse(['success' => true]); }

// Variables globales let offsetReservations = 12; // Commence après les 12 premiers let totalReservations = 0; let filtreActif = 'toutes'; // Initialisation depuis Twig function initialiserReservations(total, filtre) { totalReservations = total; filtreActif = filtre; updateCompteur(); } // Chargement AJAX async function chargerPlusReservations() { const btn = document.getElementById('btn-charger-plus'); btn.disabled = true; btn.textContent = 'Chargement...'; try { const url = `/dashboard/locataire/reservations/charger-plus?offset=${offsetReservations}&filtre=${filtreActif}`; const response = await fetch(url); const data = await response.json(); if (data.success) { // Insérer le HTML avant le bouton const container = document.getElementById('reservations-container'); container.insertAdjacentHTML('beforeend', data.html); // Mettre à jour l'offset offsetReservations = data.offset; // Cacher le bouton si plus de données if (!data.hasMore) { btn.style.display = 'none'; } updateCompteur(); } } catch (error) { console.error('Erreur:', error); } finally { btn.disabled = false; btn.textContent = 'Charger plus'; } } // Mise à jour du compteur function updateCompteur() { const compteur = document.getElementById('compteur-reservations'); const affichees = document.querySelectorAll('.reservation-card').length; compteur.textContent = `${affichees} réservations affichées sur ${totalReservations}`; }
