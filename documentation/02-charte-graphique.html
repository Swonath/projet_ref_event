1. Identité Visuelle

Philosophie du Design

La charte graphique de Références Événements repose sur une esthétique minimaliste et professionnelle. Le design privilégie la sobriété avec une palette de couleurs neutres dominée par le noir et les nuances de gris, permettant aux contenus (photos d'emplacements, données) de rester au premier plan.

Mots-clés du design : Professionnel, Épuré, Moderne, Accessible, Fiable

Valeurs transmises

  • Confiance - Couleurs sobres et professionnelles
  • Clarté - Hiérarchie visuelle claire, espaces aérés
  • Modernité - Design flat, coins arrondis, animations subtiles
  • Efficacité - Interface intuitive, actions visibles

8. Responsive Design

Breakpoints

Appareil Breakpoint Adaptations
Desktop Large > 1200px Layout complet, sidebar fixe
Desktop 992px - 1200px Grids 3 colonnes → 2 colonnes
Tablet 768px - 991px Sidebar cachée, hamburger menu
Mobile < 768px Layout 1 colonne, navigation bottom

Code CSS Media Queries

/* Tablet */ @media (max-width: 1200px) { .stats-grid { grid-template-columns: repeat(2, 1fr); } } /* Mobile */ @media (max-width: 768px) { .sidebar { transform: translateX(-100%); } .main-content { margin-left: 0; } .stats-grid { grid-template-columns: 1fr; } }
